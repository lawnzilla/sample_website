Important HTML Tags

This page is designed as a quick referencr for some of the common HTML tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the mosy important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header includes a container for introductory content
ul unordered
list		 unnumbered list tag <ul>
<fruits</li>
<li>veggies</li>
</ul>
  • fruits
  • veggies
ol ordered
list		 numbered list tag <ol>
<>cookies</li>
<>cream</li>
<ol>
  1. cookies
  2. cream
div division group elements to format them with CSS

Inline Elements

Tag Name Purpose Example Result
em make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong>
span span group inline elements <Call me Ishmael.</span> Call me Ishmael.
a anchor make hyperlink <a href="http://learnenough.com/>Learn Enough</a >
code code syntax Show code syntax <code>img</code> img
li list list element <ul> </ul>
  • warmth
  • shelter
